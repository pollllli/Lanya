# 器件管理系统

## 概述

器件管理系统是一款基于 React Native + Expo 开发的移动应用，用于管理电子器件库存和BOM配单。系统支持通过蓝牙向下位机发送指令，控制器件架上的指示灯，方便器件的请求和取出操作。

## 应用功能

### 1. 用户管理
- **登录和注册**：支持普通用户和管理员登录，注册时可选择用户角色
- **密码修改**：用户可以在个人中心修改密码
- **登录状态持久化**：应用重启后保持登录状态，无需重新登录

### 2. 器件管理
- **器件库存**：查看所有器件的详细信息，包括名称、功能、电气参数等
- **器件搜索**：支持按名称、功能、编号等多种方式搜索器件
- **器件筛选**：按器件架（A/B/C/D）筛选器件
- **器件详情**：查看器件的详细信息
- **器件上架**：管理员可以添加新器件到库存
- **器件编辑**：管理员可以编辑现有器件信息
- **器件删除**：管理员可以删除不需要的器件
- **批量导入**：支持从Excel/CSV文件批量导入器件数据

### 3. BOM配单
- **导入BOM**：从Excel/CSV文件导入BOM配单，自动识别器件
- **加载BOM**：加载已保存的BOM配单
- **删除BOM**：管理员可以删除不需要的BOM配单
- **器件状态显示**：自动识别BOM中的器件，在器件架中有库存的显示为绿色，无库存的显示为黄色
- **器件请求**：点击绿色器件时，通过蓝牙发送指令到下位机，控制对应位置的灯亮起

### 4. 器件请求与取出
- **请求器件**：发送指令点亮对应位置的指示灯
- **取出器件**：发送指令熄灭对应位置的指示灯
- **控制所有灯**：支持一键点亮或熄灭所有指示灯

### 5. 数据管理
- **数据备份**：将所有数据备份到本地
- **数据恢复**：从备份数据恢复应用数据
- **数据验证**：确保器件编号的唯一性

### 6. 连接功能
- **蓝牙连接**：连接到蓝牙设备（如CH9140）
- **串口连接**：通过USB串口连接到下位机

## 使用方法

### 1. 登录和注册
- 打开应用后，首先进入登录页面
- 输入用户名和密码登录（默认账号：admin/admin，user/user）
- 注册新用户时，选择用户角色（普通用户/管理员）

### 2. 器件管理
- **查看器件**：在"库存"标签页查看所有器件
- **搜索器件**：在搜索框中输入关键词搜索器件
- **筛选器件**：点击器件架选择器筛选特定器件架的器件
- **上架器件**：点击"上架器件"按钮添加新器件
- **编辑器件**：点击器件卡片进入详情页面，点击"编辑"按钮修改器件信息
- **删除器件**：长按器件卡片或进入编辑页面后删除器件
- **批量导入**：在"上架器件"页面点击"从Excel/CSV导入"按钮导入器件数据

### 3. BOM配单
- **导入BOM**：在"BOM配单"标签页点击"导入 BOM 文件"按钮，选择Excel或CSV文件导入BOM配单
- **加载BOM**：点击"加载已保存的 BOM"按钮选择并加载已保存的BOM
- **查看器件状态**：导入或加载BOM后，系统会自动识别器件状态，绿色表示在器件架中有库存，黄色表示无库存
- **请求器件**：点击绿色器件时，系统会通过蓝牙发送指令到下位机，控制对应位置的灯亮起
- **删除BOM**：在BOM列表中点击"删除"按钮删除BOM

### 4. 器件请求与取出
- **连接设备**：在"连接"标签页搜索并连接蓝牙设备或串口设备
- **请求器件**：在器件详情页面点击"请求此器件"按钮，点亮对应位置的指示灯
- **取出器件**：在器件详情页面点击"取出器件"按钮，熄灭对应位置的指示灯
- **控制所有灯**：在器件列表页面底部点击"点亮所有灯"或"熄灭所有灯"按钮

### 5. 数据管理
- **数据备份**：在"个人中心"页面点击"数据备份"按钮备份数据
- **数据恢复**：在"个人中心"页面点击"数据恢复"按钮恢复数据

## 权限说明

- **普通用户**：只能查看器件和BOM配单，不能修改数据
- **管理员**：可以添加、编辑、删除器件和BOM配单，导入数据

## 技术说明

- **开发框架**：React Native + Expo
- **数据存储**：AsyncStorage（本地存储）
- **蓝牙通信**：react-native-ble-plx
- **串口通信**：SerialPortHandler
- **文件处理**：Expo DocumentPicker
- **Excel/CSV导入**：xlsx库
- **指令帧构建**：自定义命令帧协议（见 PROTOCOL.md）

## 安装步骤

### 开发版
1. 克隆项目到本地
2. 安装依赖：`npm install`
3. 启动开发服务器：`npm run android`
4. 使用Expo开发客户端运行应用

### 发布版
1. 下载 `app-release.apk` 文件
2. 直接安装到Android设备上
3. 打开应用即可使用（无需开发服务器）

## 注意事项

- 批量导入Excel/CSV文件时，确保文件格式正确，包含必要的字段
- 蓝牙连接功能需要设备支持蓝牙，并授予应用蓝牙权限
- 数据备份功能仅保存到本地，建议定期导出备份数据到其他存储介质
- 串口通信需要USB转串口驱动支持（如CH340驱动）

## 命令帧格式

所有指令帧遵循以下格式（7字节）：
```
[55] [AA] [CMD] [LEN] [DATA_H] [DATA_L] [CRC]
  ↑    ↑    ↑     ↑      ↑       ↑       ↑
帧头  帧头 命令字 数据长度 数据高字节 数据低字节 CRC-8/MAXIM
```

### 支持的命令

| 命令字 | 功能 | 数据格式 |
|-------|-----|---------|
| 0x00 | 心跳 | uint16_t (0x0001) |
| 0x01 | 点亮对应灯 | uint16_t (灯ID 1-255) |
| 0x02 | 熄灭对应灯 | uint16_t (灯ID 1-255) |
| 0x03 | 控制所有灯 | uint16_t (0xFFFF=点亮, 0x0000=熄灭) |

### 响应帧格式

响应帧命令字 = 发送命令字 + 0x80

| 命令字 | 响应命令字 |
|-------|----------|
| 0x00 | 0x80 |
| 0x01 | 0x81 |
| 0x02 | 0x82 |
| 0x03 | 0x83 |
